Веб-приложение для заполнения справочника о страховых организациях(ИНН, ОГРН, Полное наименование и адрес)
Для развертывания приложения необходимо выполнить следующие операции:
 
   
1. Разархивируйте файл в отдельную папку.
   
2. Перейдите в папку, в которой находится файл pom.xml.
 
3. Введите команду для разворачивания и инициализации базы данных       mvn flyway:migrate 

4. Для запуска приложения введите команду                               mvn spring-boot:run
  
5. Перейдите в браузере по адресу http://localhost:8080/
